This is a vibe-coded frontend for the [fitnotes Android app](https://www.fitnotesapp.com/) which is backed by a sqlite store (uses a .fitnotes suffix when you export it from the app). It enables management of routines and exercises. It hasn't been manually tested extensively so if you want to use it, always back up your fitnotes data beforehand. It also only has features that I use.

To use it:

1. Export your *.fitnotes file on mobile
2. Run `npm run dev`
3. Go to the localhost site, then import your *.fitnotes file from step 1
